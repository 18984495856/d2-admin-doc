# 借鉴 D2Admin 的项目

以下开源项目中借鉴了 D2Admin 代码。

本页面不明确指出哪些是借鉴哪些是抄袭，由读者自行判断。

本页面作用是指明哪些东西是 D2Admin 原创的，避免大家看到和其它项目有类似设计时发生误会以为 D2 抄袭他人（误会发生过）。

TIP

图片右键在新标签页打开查看大图

此页面截图都会标注日期，只新增，不删除，只代表当时日期的状态
