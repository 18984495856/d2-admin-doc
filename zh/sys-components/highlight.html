# d2-highlight

代码高亮显示组件

# 参数

参数名 介绍 必选 值类型 可选值 默认值
code 代码字符串 非 String
format-html 是否格式化 HTML 非 Boolean false
lang 首选语言 非 String

TIP

format 建议只在内容为 HTML 的时候打开，如果内容不是 HTML，设置为 true 会有负面效果

# 示例

<d2-highlight code="alert('Hello')"/>

TIP

本框架只是提供代码高亮的简单实现，如需实现更高级的设置请修改组件代码

# 语言

如果不指定语言，默认会依次尝试以下语言：

  1. html
  2. javascript
  3. json
  4. css
  5. scss
  6. less
