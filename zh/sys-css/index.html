            # CSS 实用类

            # 样式

            类名 效果
            .d2-card 使 el-card 组件样式跟随主题变化
            • .d2-card 示例
            <el-card shadow="never" class="d2-card">
  <template slot="header">header</template>
  body
</el-card>
<!-- 快看 我的边框和 header 边框会随着主题切换样式！ -->

            # 文字

            类名 效果
            .d2-text-center 文字水平居中

            # 浮动

            类名 效果
            .d2-fl 左浮动
            .d2-fr 右浮动
            .d2-clearfix 清除浮动
