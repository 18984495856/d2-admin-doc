D2 Admin

优雅的管理系统前端集成方案

快速上手 →

Vue驱动

享受 Vue + webpack 的开发体验，超快虚拟 DOM 和最省心的优化。

强大的数据持久化方案

内部封装了多种数据持久化方式，您可以在数据持久化时区分用户，区分页面，甚至可以创建页面数据快照。

模块化全局状态管理

D2Admin 的系统管理为独立模块，在此模块内根据功能划分为十几个模块，每个模块都分工明确。

贴心小工具

为您准备了实用的 JavaScript Util 以及 CSS 常用类。

丰富集成

已经为你准备好了很可能用到的插件和组件，专为 D2Admin 设计，或者来自第三方。

ElementUI组件库

集成完善且强大的 ElementUI，随意搭配使用任何 ElementUI 组件。

可视化构建

支持 CLI3 的可视化面板，并且提供飞冰版本脚手架。

文档完善

提供事无巨细的文档说明，并且文档也一直在更新完善。

活跃交流群

提供QQ和微信群供大家集中讨论学习，或者互相解决问题。
